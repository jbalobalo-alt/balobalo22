# Personal Website Setup Notes

## Images Required

Please add the following images to the `public/` directory:

- `profile.jpg` - Your profile picture (200x200 recommended)
- `about.jpg` - A photo for the about page
- `hiking.jpg` - Photo for hiking hobby
- `photography.jpg` - Photo for photography hobby
- `gaming.jpg` - Photo for gaming hobby
- `cooking.jpg` - Photo for cooking hobby
- `reading.jpg` - Photo for reading hobby
- `coding.jpg` - Photo for coding projects hobby

## Personal Information

Update the following files with your actual information:

- `app/page.tsx` - Home page content
- `app/about/page.tsx` - About section
- `app/education/page.tsx` - Education details
- `app/hobbies/page.tsx` - Hobbies content
- `app/contact/page.tsx` - Contact information

## Running the Site

Run `npm run dev` to start the development server.
